转载： [IDEA破解 2017 IDEA license server 激活（可用）](http://blog.csdn.net/zhangwenwu2/article/details/54948959)

进入ide主页面:

`help` 》`register` 》`license server`

然后输入 `http://idea.iteblog.com/key.php`

点击下 `discover server`,查看下是否通，如果通的话

点击 `active` 激活

（注意：php要小写）即可~
